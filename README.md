🎵 Playlist Manager using Linked List (C++)
**SINGLY LINKLIST PLAYLIST**** 
📌 Project Description**

This project is a Playlist Management System developed using C++ and the concept of a Singly Linked List. It allows users to create and manage a music playlist dynamically through a simple and interactive menu-driven interface.

The application stores song details like Song ID, Artist Name, and Song Title, and performs various operations efficiently using linked list data structure.

🚀 **Features**
➕ Insert songs:
At the beginning
At the end
At a specific position
❌ Delete songs:
From the beginning
From the end
From a given position
🔍 Search songs:
Search songs by artist name
📋 Display playlist:
View all songs in sequence
🔢 Count songs:
Shows total number of songs available
🎯 Auto-generated Song ID system
🧠 **Concepts Used**
Object-Oriented Programming (OOP)
Classes and Objects
Pointers
Dynamic Memory Allocation
Singly Linked List
Menu-driven Programming
🏗️ Project Structure
🔹 NODE Class

**Represents each song node:**

ID → Unique Song ID

🔹 LinkList Class

**Handles all playlist operations:**

Counting nodes**

**COMBINATION OF BOTH MODULE IN ONE FILE** 
his project is a Music Playlist Management System developed using C++, based on the concepts of both Singly Linked List and Doubly Linked List.

The system provides a menu-driven interface where users can choose between singly and doubly linked list implementations to manage their playlist. Each song is stored as a node containing details such as Song ID, Artist Name, and Song Title.

This project demonstrates how data structures can be used to efficiently organize and manage dynamic data in a real-world application like a music playlist.

🚀 Features
🎯 Select implementation:
Singly Linked List
Doubly Linked List
➕ Add songs to playlist
❌ Delete songs from playlist
🔍 Search songs by artist name
📃 Display playlist
🔄 Forward and backward traversal (in doubly linked list)
🔢 Count total songs
